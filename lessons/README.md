# uCode1 Curriculum Lesson Plans

This directory contains structured lesson plans for the uCode1 education
program. Lessons are organised into four levels covering programming
concepts, computer science, and retro computing.

## Level Structure

| Level | Ages | Focus |
|-------|------|-------|
| Level 1 | 8-12 | Introduction to Programming |
| Level 2 | 10-14 | Building Skills |
| Level 3 | 12-16 | Advanced Concepts |
| Level 4 | 14-18 | Retro Computing |

## Lesson Format

Each lesson includes:
- **Title** and **topic**
- **Learning objectives**
- **Explanation** with code examples
- **Interactive exercises**
- **Quiz** to test understanding
- **Summary** of key points

## Loading Lessons

In uCode1 education mode:

```bash
EDUCATE        # Enter education mode
LESSON 1       # Load lesson 1
NEXT           # Next lesson
BACK           # Previous lesson
QUIZ           # Take the quiz
SCORE          # View progress
```
