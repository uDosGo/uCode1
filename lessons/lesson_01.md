# Lesson 1: What is a Computer?

**Level:** 1 — Introduction to Programming
**Age Group:** 8-12
**Duration:** 30 minutes

## Learning Objectives

By the end of this lesson, students will be able to:
- Identify the four main parts of a computer
- Understand input vs output
- Turn on uCode1 and use basic commands

## Explanation

A computer has four main parts:

1. **Input** — things you use to tell the computer what to do
   (keyboard, mouse, joystick)
2. **Processing** — the brain of the computer that thinks
   (the CPU / microprocessor)
3. **Memory** — where the computer stores information
   (RAM, storage)
4. **Output** — how the computer shows you results
   (screen, speaker, printer)

### The uCode1 Computer

uCode1 is like a BBC Micro computer from the 1980s. It has:
- A keyboard for **input**
- A 6502 microprocessor for **processing**
- Memory for storing programs
- A screen for **output**

## Interactive Exercise

1. Turn on uCode1 (or start the emulator)
2. Type `HELP` and press Enter
3. Type `VER` to see the version
4. Type `STATUS` to see system status
5. Type `TIME` to see the current time

## Quiz

1. Which part of the computer is the "brain"?
   a) Keyboard
   b) CPU
   c) Screen
   **Answer: b) CPU**

2. Is a keyboard input or output?
   a) Input
   b) Output
   **Answer: a) Input**

3. What does `VER` do?
   a) Shows the version
   b) Clears the screen
   c) Runs a program
   **Answer: a) Shows the version**

## Summary

- Computers have input, processing, memory, and output
- uCode1 is a retro computer emulator
- You can type commands to talk to the computer
- `HELP` shows available commands
